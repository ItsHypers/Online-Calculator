<h3>Keyboard Commands:</h3>

I have added in keyboard commands, so you can use your keyboard to input operations and numbers.

<ul>

<li> + = + </li>
<li> - = -</li>
<li> * = *</li>
<li> % / ÷ = ÷</li>
<li> = Enter = Enter</li>
<li> Number Key = Number</li>
<li> ; = ** (^)</li>
<li> # = +/-</li>
<li> . = .</li>
</ul>
